## Usage:

- **web /index-kecurangan.php:**  
    - Menampilkan data TPS Bermasalah

- **web /index.php:**  
    - Menampilkan real count dari KPU

- **cli /get-all-tps.php:**  
    - Mengunduh semua TPS yang ada di KPU

- **cli /cek-kecurangan/main.php:**  
    - Mengecek dan menghitung selisih suara di semua TPS
